ScatterTangram
==============

Project developped in 2010. 

This project was to create a simple puzzle game on the Microsoft Surface table, the aim was to test the possibilities of the Microsoft Surface table and to do some user study of the interaction with children.

Développement d'un jeu de puzzle plus précisément de Tangram sur la plateforme tactile Microsoft Surface. Le but était de tester les possibilitées de la table Microsoft Surface et d'étudier les intéractions entre des enfants et ce jeu.
